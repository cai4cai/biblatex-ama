# biblatex-ama

BibLaTex style files for [AMA style](https://libguides.jcu.edu.au/ama).

## To Use

Add the files `ama.bbx` and `ama.cbx` to your project.

Include the following in your preamble:
```latex
\usepackage[citestyle=ama, bibstyle=ama]{biblatex}
```
